HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Parent/Child elements represent the outer elements (Parents) and the inner nested elements (children)
  • Metadata is contained in the Meta tag, one important one is the viewport, this makes your page resize.
  • HTML is the framework for a solid site.
  • HTML makes for a good user experience by organizing different types of data
  • The Browser reads metadata and this can be useful for SEO using the title
