HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Parent/Child elements represent the outer elements (Parents) and the inner nested elements (children)
  • Metadata is contained in the Meta tag, one important one is the viewport, this makes your page resize
  • HTML is the framework for a solid site
  • HTML makes for a good user experience by organizing different types of data
  • The Browser reads metadata and this can be useful for SEO using the title

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Adds color and style
  • Stands for cascading style sheets
  • Three types of css are inline css, internal css style sheet, and external css style sheet
  • The Most common css style is external
  • There is a selector, and a declaration block (in braces{}) with different properties (followed by :) and values (followed by ;)
  • Multiple selectors can be used with one declaration block if they have the same formatting style

Git

  • Git is used for version control when developing software.
  • Version control is accomplished with git commits frequently
  • This works by using different branches, this way you do not change anything on the live site.
  • Git commands are done in the terminal. We set up SSH keys in order to make it so you don't have to log in every time
  • Git helps developers when debugging by providing a record of when certain changes were made
  • Github is a social network centered around the language Git, it is often used for collaboration between developers on projects

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Functions are used to execute a block of code when the program calls for it
  • JavaScript adds interactivity to the website, like the house analogy
  • JavaScript will be used for half of the bootcamp, so it is important to understand
  • Some of this will be confusing now, but will make a lot of sense in the future after using it, Don't be discouraged!
  • The Chromedev features are really handy for seeing your code live, as well as debugging!
